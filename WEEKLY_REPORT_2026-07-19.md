# iOS 端周报 - 2026-07-19

## 周目标是什么

本周按《BNBU Sports Week 3 三端开发任务书》推进，iOS 端聚焦：

- **W3-MOB-01（P0）**：以 Android 学生端 Compose 源码为前端结构基线，对齐导航、页面层级、组件、主题和主要操作路径。

## 遇到哪些困难，踩到了哪些坑

1. **Android 与 iOS 后端合同不完全一致**：Android 源码仍含补材料端点；iOS 按合同与产品边界保留“已提交”与驳回后重新申请，不盲目照抄。
2. **后端迭代很快**：周中从旧 3333/96 切到 `:82`，今日再升到 r19，业务错误码与字段（如 `studentNumber`、`cosKey`）持续变化；客户端必须做降级兼容，不能假设所有工作台接口永远 200。
3. **反馈包不能原样编译**：`Models.swift` 解码链缺少 `try?`，以及旧单测与新的 1h/2h + 可解析截止时间规则冲突，需同步修测试。
4. **系统“保存密码？”弹窗**会挡住 UI 自动化切 Tab，联调测试需额外处理。
5. **每日一次打卡限制**曾挡住跨日闭环复测，需换日或确认当日无记录后再验提交。
